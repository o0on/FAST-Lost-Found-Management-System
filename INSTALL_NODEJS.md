# Node.js Installation Guide (Urdu/Hindi)

## Server Start Karne Ke Liye Node.js Install Karein

### Step 1: Node.js Download Karein
1. Browser mein jayein: **https://nodejs.org/**
2. **LTS version** download karein (recommended)
3. `.msi` file download hogi

### Step 2: Install Karein
1. Downloaded file ko run karein
2. "Next" buttons pe click karte rahein
3. Default settings theek hain - "Install" pe click karein
4. Installation complete hone ka wait karein

### Step 3: Terminal Restart Karein
- PowerShell ya Command Prompt ko **completely close** karein
- Naya terminal open karein

### Step 4: Verify Karein
```powershell
node --version
npm --version
```
Agar version numbers dikhein, to installation successful hai!

### Step 5: Project Run Karein
```powershell
cd "C:\Users\uh723\OneDrive\Pictures\fast-lost-found"
npm install
npm run dev
```

Server automatically browser mein khul jayega!
